Thinking Like a Programmer

Procedural Thinking

- Every step has to be clearly/logically described, leaving no room for ambiguity.

Abstract Thinking

- Finding generality amongst seemingly different things.

Technological Empathy

- Understand what the computer is/how it works.

- What it is good and bad at doing.

Debugging

- Collect evidence.

- Generate theories.

- Test those theories.

- Fix the problem.

What is HTML?

- It is a markup language for creating web pages. HTML stands for Hyper Text Markup Language.

- Opening and closing tags with content in between result in HTML elements.

- A webpage contains several elements that are rendered by the browser.

What is CSS?

- CSS is used to enhance the appearance of a webpage. It is the way the website will appear to the user.

- "Cascading" means that 'rules are applied not only to the elements they directly match, but also to all of those elements' child elements.

- However, if a child element has multiple, overlapping rules defined for it, the more specific rule takes effect.'

Summary of Notes

  • An Html page is the rendition of several elements by a browser, made flashy/presentable by CSS.
